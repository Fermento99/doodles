# Doodles
Repository aggregating small visual projects

## Scripts
- `npm run dev %sub-project-name%` - run development server a sub project
- `npm run build %sub-project-name%` - create a production build just for one a sub project
- `npm run build-all` - create a production build for the whole site
- `npm run new-side %sub-project-name%` - create a new sub project from template
- `npm run deploy` - deploy project to GitHub-pages
